# House Sales in King County, USA

This repository contains a Tableau dashboard project that visualizes house sales data. The dashboard provides detailed insights into various aspects of the housing market, including price trends, geographical distribution, and property features.

## Overview

### Key Features

- **Daily Average House Sales Price:** Line chart displaying the average sales price of houses on a daily basis for July 2014.
- **Map Visualization:** A geographical map showing the distribution of house sales prices across different regions.
- **Distribution of House Prices:** Histogram depicting the frequency distribution of house prices.
- **Distribution of Bathrooms and Bedrooms:** Histograms showing the frequency distribution of the number of bathrooms and bedrooms in sold houses.
- **View vs Condition Heatmap:** A heatmap correlating house views (e.g., excellent, good, average) with their condition and average prices.
- **Interactive Filters:** Allows users to filter data by month, year built, square footage, and more to customize the visualizations.

## Getting Started

### Prerequisites

- Tableau Desktop or Tableau Public

### Installation

1. Clone this repository to your local machine:
    ```sh
    git clone https://github.com/your-username/house-sales-dashboard.git
    ```
2. Open Tableau and load the `.twbx` file from the cloned repository.

### Usage

1. Explore the dashboard to gain insights into house sales data.
2. Use the interactive filters to customize the view based on your requirements.
3. Analyze trends, distributions, and correlations within the housing market.

## Project Structure

- `HouseSalesDashboard.twbx`: The Tableau workbook file containing the dashboard.
- `README.md`: This file.
- `Screenshots/`: Directory containing screenshots of the dashboard.
